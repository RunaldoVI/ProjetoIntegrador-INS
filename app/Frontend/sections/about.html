Sobre o Projeto

O Projeto Integrador INS é uma iniciativa acadêmica interdisciplinar que visa transformar conhecimentos teóricos em soluções práticas. A plataforma foi desenvolvida para realizar a ingestão, análise e visualização de arquivos PDF de forma intuitiva, eficiente e acessível.

🎯 Missão

Aplicar conhecimentos acadêmicos para desenvolver uma solução funcional que otimize o manuseio e a análise de documentos digitais.

🌟 Visão

Ser um projeto referência em integração de competências técnicas em ambientes educacionais, com impacto positivo e aplicabilidade real.

⚙️ Funcionalidades

  • 📥 Upload e processamento inteligente de arquivos PDF
  • 🌓 Interface moderna com suporte a tema claro e escuro
  • 🕓 Histórico de documentos ingeridos
  • 🧠 Simulação de análise semântica com retorno visual
  • 📊 Exportação de resultados estruturados (formato Excel)
  • 🔐 Acesso autenticado com área de perfil

💻 Tecnologias Utilizadas

O sistema foi desenvolvido com foco em desempenho, escalabilidade e experiência do utilizador, utilizando:

  • React e Tailwind CSS (Frontend)
  • Node.js e Express (API)
  • MySQL (Base de dados relacional)
  • Ollama + LLM Mistral (Processamento semântico)
  • Docker e Docker Compose (Orquestração de containers)
